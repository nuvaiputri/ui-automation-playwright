# UI Automation Framework - Playwright (JavaScript)

## 📌 Overview
This repository contains a UI automation framework built using Playwright with JavaScript.

It demonstrates:
- UI test automation best practices
- Page Object Model structure
- Clean and scalable project architecture
- CI integration (coming soon)

---

## 🛠 Tech Stack
- Playwright
- JavaScript (ES6+)
- Node.js
- GitHub Actions (CI - planned)

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository
git clone https://github.com/nuvaiputri/ui-automation-playwright.


### 2️⃣ Install Dependencies
npm install


### 3️⃣ Install Playwright Browsers
npx playwright install


---

## ▶ Run Tests

### Run all tests
npx playwright test


### Run tests in UI mode
npx playwright test --ui


### Run specific test file
npx playwright test tests/homepage.spec.js


---

## 📂 Project Structure
tests/ → Test cases
pages/ → Page Object Model (planned)
utils/ → Helper functions (planned)
playwright.config.js → Framework configuration
